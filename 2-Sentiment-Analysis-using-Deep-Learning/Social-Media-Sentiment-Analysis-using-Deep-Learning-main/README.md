# Social Media Sentiment Analysis using Deep Learning

## Project Overview
This project performs sentiment analysis on social media tweets using Natural Language Processing (NLP) and Deep Learning. The model classifies tweets as positive or negative.

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- NLP (Tokenization & Padding)

## Dataset
Twitter Sentiment Dataset containing tweets labeled as positive or negative.

## Model Architecture
- Tokenization
- Sequence Padding
- Embedding Layer
- LSTM Layer
- Dense Output Layer

## Results
- Initial Accuracy: 95%
- Balanced Dataset Accuracy: 83.38%

The dataset was highly imbalanced initially, so undersampling was applied to balance the classes.

## Output
The trained model can predict whether a tweet expresses positive or negative sentiment.
